Description of method

—Mass Balances:

[NH3]Total = [NH3] + [NH4+] + 4[Ni(NH3)42+] + 6[Ni(NH3)62+]

[NiII]Total = [Ni2+] + [Ni(OH)2] + [Ni(NH3)42+] + [Ni(NH3)62+]+[NiCit-] + [NiHCit] + [NiH2Cit+]

[Citrate]Total = [H3Cit] + [H2Cit-] + [HCit2-] + [Cit3-] + [NiCit-] + [NiHCit] + [NiH2Cit+]

—Chemical reactions:

1. NH3(g) = NH3(aq)

2. NH4+ + H+ = NH3

3. Ni(OH)2(c) + 2H+ = Ni2+ + 2H2O

4. Ni(OH)2 + 4NH4+ = Ni(NH3)42+ + 2H2O + 2H+

5. Ni(OH)2 + 6NH4+ = Ni(NH3)62+ + 2H2O + 4H+

6.Ni(OH)2(c) + 4NH3 + 2H+= Ni(NH3)42+ + 2H2O

7. Ni(OH)2(c) + 6NH3 = Ni(NH3)62+ + 2H2O

8. H3Cit = H+ + H2Cit-

9. H2Cit- = H+ + HCit2-

10. HCit2- = H+ + Cit3-

11. Ni2+ + Cit3- = NiCit-

12. Ni2+ + HCit2- = NiHCit

13. Ni2+ + H2Cit- = NiH2Cit+

14. Ni(OH)2(c) + 2H+ = Ni2+ + 2H2O

15. NiCit- + 2H2O = Ni(OH)2 + 2H+ + Cit3-

16. Ni2+ + H3Cit = NiH2Cit+ + H+

17. NiH2Cit+ = NiHCit + H+

18. NiHCit = NiCit- + H+

—Electrochemical reactions:

1. Ni2+ + 2e- = Ni

2. Ni(OH)2 + 2H+ +2e- = Ni + 2H2O

3. Ni(NH3)42+ + 4H+ +2e- = Ni + 4NH4+

4. Ni(NH3)62+ + 6H+ +2e- = Ni + 6NH4+

5. Ni(NH3)42+ +2e- = Ni + 4NH3

6. Ni(NH3)62+ +2e- = Ni + 6NH3

7. Ni(OH)3 + 4NH4+ + e- = Ni(NH3)42+ +3H2O + H+

8. Ni(OH)3 + 6NH4+ + e- = Ni(NH3)62+ +3H2O + 3H+

9. Ni(OH)3 + 6NH4+ + 3H+ + e- = Ni(NH3)62+ +3H2O

10. Ni(OH)3 + 3H+ + e- = Ni2+ + 3H2O

11. Ni(OH)3 + H+ + e- = Ni(OH)2+ + H2O

12. NiH2Cit+ + H+ + 2e- = Ni + H3Cit

13. NiHCit + 2H+ + 2e- = Ni + H3Cit

14. NiHCit + H+ + 2e- = Ni + H2Cit-

15. NiCit- + 2H+ + 2e- = Ni + H2Cit-

16. NiCit- + H+ + 2e- = Ni + HCit2-

17. NiCit- + 2e- = Ni + Cit3-

18. Ni(OH)2 + 2H+ +2e- = Ni + 2H2O

19. Ni(OH)3 +H3Cit + 2H+ + e- = NiH2Cit+ + 3H2O

20. Ni(OH)3 +H3Cit + H+ + e- = NiHCit + 3H2O

21. Ni(OH)3 +H2Cit- + 2H+ + e- = NiHCit + 3H2O

22. Ni(OH)3 +H2Cit- + H+ + e- = NiCit- + 3H2O

23. Ni(OH)3 +HCit2- + 2H+ + e- = NiCit- + 3H2O

24. Ni(OH)3 +Cit3- + 3H+ + e- = NiCit- + 3H2O

25. O2 + 4H+ + 4e- = 2H2O

26. 2H+ + 2e- = H2
