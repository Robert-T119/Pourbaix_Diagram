Description of method

—Mass Balance:

[CuII]Total = [Cu2+] + [Cu(OH)2] + [HCuO2-] + [CuO22-]

[CuII]Total = [Cu2+] + [CuO] + [HCuO2-] + [CuO22-]

—Chemical reactions:

1. CuO + 2H+ = Cu2+ + H2O

2. Cu(OH)2 + 2H+ = Cu2+ + 2H2O

3. CuO + H2O = HCuO2- +H+

4. CuO + H2O = CuO22- + 2H+

5. Cu(OH)2 + H2O = HCuO2- + H+

6. Cu(OH)2 = CuO22- + 2H+

7. HCuO2- = CuO22- + H+

8. Cu2+ + 2H2O = CuO22- + 4H+

9. Cu2+ + 2H2O = HCuO2- + 3H+

10. Cu2O + 2H+ = 2Cu+ + H2O

11. 2Cu2+ = Cu2+ + Cu

—Electrochemical reactions:

1. Cu2+ + e- = Cu+

2. Cu2+ + 2e- = Cu

3. 2Cu2+ + 2H2O +2e- = Cu2O + 2H+

4. 2CuO + 2H+ +2e- = Cu2O + H2O

5. Cu2O + 2H+ +2e- = 2Cu + H2O

6. 2HCuO2- + 2H+ + 2e- = Cu2O + 2H2O

7. 2CuO22- + 6H+ + 2e- = Cu2O + 3H2O

8. CuO22- + 4H+ + 2e- = Cu + 2H2O

9. O2 + 4H+ + 4e- = 2H2O

10. 2H+ + 2e- = H2
